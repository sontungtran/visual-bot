## Getting started
### Install dependencies
#### Requirements
- PyTorch>=0.4.1
- opencv-python>=3.4.2
- check requirements.txt
```
pip install -r requirements.txt
```
#### Pretrained Models
[Text detector](https://drive.google.com/file/d/1Jk4eGD7crsqCCg9C9VjCLkMN3ze8kutZ) ([CRAFT](https://github.com/clovaai/CRAFT-pytorch))
